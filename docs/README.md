# GitHub Pages

Esta carpeta existe únicamente para publicar el informe de mercados como página web mediante **GitHub Pages**.

`index.html` es una copia exacta de [`informes-mercados/2026-07-17-informe-mercados-grafico.html`](../informes-mercados/2026-07-17-informe-mercados-grafico.html). El original en `informes-mercados/` sigue siendo la fuente de verdad; esta copia se actualiza cada vez que se publique un nuevo informe como página pública.

## Activación (paso manual, una sola vez)

GitHub Pages requiere habilitarse desde la configuración del repositorio — esto no se puede hacer vía API/commits, solo desde la interfaz de GitHub:

1. Ir a **Settings → Pages** en el repositorio.
2. En "Build and deployment" → **Source**: `Deploy from a branch`.
3. **Branch**: `main` — **Folder**: `/docs`.
4. Guardar.

La página quedará disponible en: `https://jjruiz54-svg.github.io/Asesor-Financiero/`

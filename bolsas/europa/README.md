# 🇪🇺 Bolsas de Valores de Europa

Categoría 4 del Asesor Financiero. Gestionada por el sub-agente **`bolsa-europa`** (ver `.claude/agents/bolsa-europa.md`).

**Cobertura:** Euronext, Londres (FTSE 100), Fráncfort (DAX), Madrid (IBEX 35), Zúrich (SMI) y los índices STOXX 600 y Euro Stoxx 50.

| Subcarpeta | Contenido |
|------------|-----------|
| `investigacion-activos/` | Fichas de análisis de activos europeos |
| `oportunidades-inversion/` | Tesis de inversión documentadas con escenarios y riesgos |
| `historia-instrumentos/` | Comportamiento histórico de los instrumentos (mínimo 5 años) |
| `actividades/` | Bitácora: decisiones del BCE, seguridad energética, eventos geopolíticos |

Reglas de nomenclatura, plantillas y fuentes permitidas: ver [`CLAUDE.md`](../../CLAUDE.md).

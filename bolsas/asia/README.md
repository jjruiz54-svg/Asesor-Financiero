# 🌏 Bolsas de Valores de Asia

Categoría 3 del Asesor Financiero. Gestionada por el sub-agente **`bolsa-asia`** (ver `.claude/agents/bolsa-asia.md`).

**Cobertura:** Tokio (Nikkei 225, TOPIX), Hong Kong (Hang Seng), Shanghái/Shenzhen (SSE, CSI 300), Seúl (KOSPI) y Singapur (STI).

| Subcarpeta | Contenido |
|------------|-----------|
| `investigacion-activos/` | Fichas de análisis de activos asiáticos |
| `oportunidades-inversion/` | Tesis de inversión documentadas con escenarios y riesgos |
| `historia-instrumentos/` | Comportamiento histórico de los instrumentos (mínimo 5 años) |
| `actividades/` | Bitácora: decisiones del BoJ y el PBoC, tensiones China–EE. UU., eventos geopolíticos |

Reglas de nomenclatura, plantillas y fuentes permitidas: ver [`CLAUDE.md`](../../CLAUDE.md).
